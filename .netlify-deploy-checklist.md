- [ ] `MONGODB_URI` = Your MongoDB Atlas connection string

- [ ] `JWT_EXPIRES_IN` = `7d`

- [ ] `NEXTAUTH_SECRET` = Random 32+ character string

**Generate secrets:**
```bash
openssl rand -base64 32
```
